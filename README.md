# Visio-zemfack

Application for meetings, note taking and transcription

## Start the project

First, you need to clone it localy

```bash
git clone https://github.com/ngimdock/visio-zemfack.git
```

Navigate to the root directory

```bash
cd visio-zemfack
```

Install dependencies

```
yarn install
```

Start the project

```
yarn dev
```

View the result

```
http://localhost:3000/
```
