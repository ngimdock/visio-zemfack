# Visio-zemfack

Application for meetings, note taking and transcription

## Start the project

First, you need to clone it localy

```bash
git clone https://github.com/ngimdock/visio-zemfack.git
```

## Install dependencies

```
yarn install
```

Start the project

```
yarn dev
```

## View the result

```
http://localhost:3000/
```

## Start peerjs server

```bash
peerjs --port 6001
```
